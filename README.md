# CloudCompute Marketplace

A peer-to-peer marketplace where users can rent out idle compute power and renters can lease GPUs/CPUs by the hour for ML training, rendering, and other workloads.

This repository is a **monorepo**:

- `CloudComputeMarketPlace-frontend/` — React + Vite SPA
- `CloudComputeMarketPlace-backend/` — Express + Mongoose REST API

> Software Engineering project — Spring 2026.
